Topple is a colourful, physics-powered block stacking game where your only goal is to build the tallest most wobbly tower you can. Drag, drop and stack a parade of vibrant shapes: squares, triangles, rectangles. Then watch gravity do its thing. Will your tower stand proud, or come tumbling down in a rainbow mess?

**Features**

- Drag and drop blocks to stack
- Realistic physics: blocks tumble, teeter and topple
- Quick to play


## Development

```sh
npm install
npm run dev
```


## Build

```sh
npm run build
```


## Deploy

This project uses GitHub Actions to deploy to GitHub Pages automatically on every push to `main`.

- The workflow builds the app and publishes the `dist` folder to the `gh-pages` branch.
- Make sure your repository's GitHub Pages settings are set to deploy from the `gh-pages` branch.
